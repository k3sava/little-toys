# little tools

60 fast, single-purpose web utilities. No accounts, no tracking, nothing uploads. Everything runs in your browser.

Live at **[tools.iamkesava.com](https://tools.iamkesava.com)**.

## Stack

- Next.js 14 with `output: 'export'` (fully static)
- Tailwind CSS with a custom multi-theme system (paper, brutalist, terminal, zen, editorial)
- Deployed to GitHub Pages via GitHub Actions

## Local dev

```
npm install
npm run dev
```

## Build static bundle

```
npm run build
# → out/
```
